== Description ==

Wp-Scribd-List provide you, a list of documents (whit the link to the download page) that you have added on your scribd's profile. You could use this plugin, to manage the download section of your site through Scribd, in web 2.0 style.

== Installation ==

1. Unzip the wp-scribd-list packet
2. Upload the folder wp-scribd-list on /wp-content/plugins/
3. Active the plugin through the 'Plugins' menu in WordPress
4. Go in admin options, and add yor API Key and Secret Key
5. In each post or page that you want to use it, simple add the [scribd-list] tag. That's all!
